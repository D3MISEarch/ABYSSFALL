# AbyssFall

A dark-fantasy action dungeon crawler built in Godot 4.4.1. The current playable prototype is **Void Warlock v0.4 Hotfix 3: The Sunken Crypts**—a connected level with progression, gear, traps, enemy generators, and a three-phase Hollow King boss.

## Current playable class

**Void Warlock** — controls gravity, portals, soul energy, and a living biomechanical Corruption resource.

## Planned second class

**The Penitent** — a sigil combatant who carves Brands and Circles into enemies and the battlefield, then activates them through ritual strikes and sacrifice.

## Run locally

1. Install Godot 4.4.1.
2. Import `project.godot`.
3. Press **F5**.
4. Use `PLAYTEST_CHECKLIST.md` to record runtime and balance problems.
