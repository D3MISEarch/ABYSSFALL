# ABYSSFALL — SHARED CAMPAIGN AND CHARACTER ARC DOCTRINE

Status: Approved, binding narrative architecture  
Codex version: 1.0  
Last updated: 2026-07-22

## Core rule

AbyssFall has **one shared world, one shared campaign timeline and one shared central conflict**.

Playable classes do not receive mutually incompatible replacement campaigns. Every class experiences the same catastrophe, major regions, principal hubs, campaign acts, world-state progression, primary factions, universal campaign antagonists, major campaign bosses and broad endgame reality.

Each class also receives a personal arc layered through that campaign.

The shared campaign answers:

> What is happening to the world, who is exploiting it and what must be done to prevent total collapse?

A class arc answers:

> How does this character understand, survive and personally resolve their relationship with that crisis?

These stories intersect, but they do not replace one another.

## Narrative ownership

### Shared campaign layer

Owns world history, the primary causes and consequences of AbyssFall, universal act structure, major settlements and regions, central hubs, shared faction conflicts, campaign-critical NPCs, required dungeons, universal bosses, cinematics, the central finale and the transition into endgame.

### Character arc layer

Owns class origin, mentor or specialist faction, personal rivals, class trials, mechanic-unlock quests, class interpretations of shared events, optional or milestone class dungeons, a class mastery finale, final class evolution and class-specific post-campaign hooks.

### Intersection layer

Shared missions may contain class-specific dialogue, alternate investigation interactions, shortcuts, optional objectives, environmental readings, boss reactions and personal follow-up quests. These deepen context or tactical expression without creating contradictory campaign canon.

## Universal campaign invariants

1. The world occupies one timeline.
2. Major campaign events occur for every class.
3. A universal boss cannot be canonically alive for one class and long dead for another.
4. Required regions cannot have incompatible destruction states between classes.
5. One class cannot secretly be the sole cause or sole solution of the whole campaign unless that truth is established for every class.
6. Class finales cannot erase infrastructure required by the campaign or endgame.
7. Class choices may alter personal relationships, local class spaces, dialogue and class endgame hooks without splitting the game into incompatible realities.
8. Shared lore remains true from every class viewpoint, even when classes interpret it differently.
9. Class-exclusive knowledge may reveal another angle of shared truth, but cannot make the campaign nonsensical for players who never use that class.
10. Future classes must fit the established campaign rather than demanding a replacement campaign.

## Class-specific openings

A class may have a unique opening establishing origin, immediate crisis, core mechanic, mentor or faction and the reason the character enters the shared campaign.

The opening should converge on the universal first hub or first campaign milestone quickly enough that production does not require eight separate Act Ones.

Recommended structure:

- a 10–30 minute class-specific prologue,
- a shared convergence event,
- arrival at the universal campaign hub,
- and a first class follow-up quest soon afterward.

A shared prologue with shorter class-specific introductions is also valid. The final structure belongs to the universal campaign bible, not one class document.

## Class arcs across campaign acts

Class arcs should appear between shared campaign milestones.

Example rhythm:

- Prologue: class origin
- Early Act I: mentor or class-faction contact
- Act I milestone: first class trial
- Between Acts I and II: class investigation
- Act II milestone: additional trials
- Mid-campaign: ultimate-unlock quest
- Later acts: rival escalation and class-faction consequences
- Near campaign completion or level cap: class mastery finale
- Endgame: replayable trials, unresolved rivals and advanced class systems

Class content may be optional, strongly encouraged or required for specific class unlocks. It should not halt the universal campaign indefinitely unless required for basic class functionality.

## Shared locations, different class readings

The same ruined city can remain one location and one event while revealing different layers:

- **Voidbringer:** impossible geometry, failed Meridian containment and unresolved locations.
- **Penitent:** ritual law, sacrifice records and attempts to bind the catastrophe through judgment.
- **Graftborn:** biological adaptation, harvested organs and survivors changed to endure the environment.
- **Somnarch:** memory residue, collective nightmares and dream structures left by the dead.

The class changes what details are visible, who speaks to the player, what optional objective appears and what personal follow-up becomes available—not the shared event itself.

## Shared factions and class factions

Shared factions influence the universal campaign and are encountered by all classes. A class may have unique history or dialogue with them, but their broad role remains consistent.

Class factions support personal arcs and mechanics. They may occupy a wing of a universal hub, linked sub-hub, instanced sanctuary, hidden mobile base or class-specific network.

They still exist in the same world and timeline. Other classes may hear about, encounter or assist them even when they cannot use their class systems.

## Mentors, rivals and mastery finales

Class mentors and rivals remain real world characters. They may appear in shared scenes, provide context to other classes, participate in faction conflicts or become future account-wide NPCs.

A class rival is not automatically the universal campaign villain.

Each class may receive a mastery finale with a bespoke location, ideological rival, full-kit mechanical exam, build-influenced resolution and permanent class reward. This is a pinnacle class quest or major side arc—not the universal campaign conclusion.

It may unlock Grand Capstones, final cosmetics, advanced class modifications, mastery titles, replayable trials and class-specific postgame encounters. It cannot contradict the shared ending.

## Voidbringer narrative reclassification

The existing Voidbringer narrative remains approved under this classification.

### Voidbringer-specific class arc

- the player’s Integrated Manifold Harness,
- the Reference Body problem,
- the Last Measure as a class-specialist faction,
- the Gaugehouse as a class hub or linked sub-hub,
- Sable Marr,
- Hadrik Coil,
- Ilyra Nham,
- Edras Vey,
- the Unmeasured,
- the Displaced,
- Voidbringer-specific enemy families,
- the eight Manifold Trials,
- the Dead Star unlock quest,
- the Axis Vault assault,
- and the confrontation with Edras.

### Shared-world material

- the Black Measure,
- spatial scars,
- Meridian Vault ruins,
- failed containment sites,
- unresolved locations,
- displaced survivors,
- and evidence that Manifold technology contributed to or attempted to contain parts of AbyssFall.

Other classes can encounter these phenomena without gaining Voidbringer controls or interpretation.

### Edras and the Axis Vault

Edras Vey is the Voidbringer mastery antagonist, not automatically the universal final villain.

His Stillpoint doctrine resolves the Voidbringer’s personal thematic conflict: control versus freedom, containment versus domination and whether one Reference Body should decide reality for everyone.

The Axis Vault is a class mastery finale woven into the shared timeline. It may occur before the universal finale, after campaign completion as a level-cap quest or at another milestone chosen by the shared campaign structure.

Its resolution must preserve the universal campaign and endgame.

## Voidbringer ending constraints

Architect, Redshift, Hollow and hybrid resolutions remain approved as class-arc resolutions.

They may alter Edras’s fate, the local Axis Vault state, the Harness, Sable’s future, the Gaugehouse, class dialogue and Voidbringer endgame hooks. They do not create incompatible global campaigns.

- **Architect:** the sealed Stillpoint remains a localized class threat or research burden. It cannot become the sole world center or invalidate endgame geography.
- **Redshift:** the Stillpoint and relevant Axis machinery are scattered or destroyed locally. This cannot remove universal travel, regions or infrastructure required by all classes.
- **Hollow:** the player carries a dormant Stillpoint fragment. This creates class-specific danger and hooks, not unlimited canonical control over the world.
- **Hybrid:** conclusions alter class presentation and personal consequences without splitting universal world state.

## Multiplayer and party rules

Class arcs must support mixed-class parties.

Recommended behavior:

- the quest owner receives primary class dialogue,
- party members may assist in combat,
- non-owner classes receive contextual dialogue rather than false class ownership,
- class unlock rewards go only to eligible characters,
- shared loot and campaign rewards follow normal party rules,
- completion flags are awarded only where narratively and mechanically valid.

A Penitent assisting in a Manifold Trial does not become a Voidbringer. A Voidbringer assisting in a Penitent rite does not gain Penitent progression.

## Quest-state separation

Narrative design distinguishes:

### Shared campaign state

Act completion, universal boss defeat, shared hub progression, region state, campaign faction state and endgame unlock.

### Character-class state

Class Trial completion, ultimate unlock, mentor relationship, mastery-antagonist resolution, class visual evolution and mastery title.

### Account-level discovery state

Potentially lore entries, Codex pages, cosmetics and cinematic replay, subject to approved architecture.

Exact persistence ownership must align with RuntimeSession, BuildData, save ADRs and the Engineering Constitution. This document defines narrative separation; it does not authorize new persistence architecture.

## Shared campaign authority

Until a universal campaign bible is approved:

- no character Codex may declare its class antagonist the universal final villain,
- no class finale may define the only canonical ending of AbyssFall,
- no class document may permanently destroy a required campaign region,
- no class-specific explanation may become the sole confirmed cause of AbyssFall without project-level approval.

The future shared campaign bible will own the universal act structure, central antagonist, ending, regions, faction chronology and endgame transition.

## Requirements for future class Codices

Every class narrative must explicitly separate:

1. Shared campaign intersections
2. Class-specific origin
3. Class faction and mentor
4. Class trials and mechanic unlocks
5. Class interpretation of shared regions
6. Class mastery antagonist
7. Class mastery finale
8. Local consequences
9. Shared-campaign constraints
10. Multiplayer participation rules
11. Postgame class hooks

It must answer:

- Where does the class join the shared campaign?
- Which shared events reveal unique class context?
- Which content is exclusive, and why?
- Can other classes assist?
- Which consequences are personal, local, shared or account-wide?
- Does any class resolution accidentally overwrite the universal campaign?

## Non-negotiable narrative rules

- One world, one campaign timeline.
- Different journeys, not different universes.
- Class depth comes from perspective, relationships and personal consequences.
- Shared events remain coherent from every class viewpoint.
- Class arcs may be emotionally complete without becoming replacement campaigns.
- Class finales test full class mastery.
- The universal finale remains meaningful for every class.
- Class-specific lore enriches world lore rather than partitioning it.
- No future class requires rebuilding the entire campaign from scratch.

> **Every class walks through the same falling world. What changes is what they can perceive, what they are forced to carry and which personal reckoning waits along the road.**
