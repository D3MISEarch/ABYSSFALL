# ABYSSFALL — VOIDBRINGER ITEMIZATION BIBLE
## Manifold Frames, Class Equipment, Affix Pools, Legendary Effects and Unique Artifacts

Status: Approved  
Codex version: 1.0  
Implementation status: Design approved  
Last updated: 2026-07-22  
Canonical class ID: `voidbringer`

---

# 1. Itemization promise

Voidbringer equipment must accomplish three things:

1. **Strengthen the player’s chosen build**
2. **Change how spatial mechanics behave**
3. **Create new build ideas the skill tree cannot produce alone**

Loot should never reduce to:

> This item has 4% more damage, so the previous item is trash.

Raw statistics still matter, but the memorable choice should be:

> Do I want my anchors fixed to terrain, orbiting my body, buried inside corpses or fused into one impossible center?

A good Voidbringer item changes one of these relationships:

- What can become an anchor
- How Mass is generated
- How Mass is released
- How Fold Lines behave
- What happens during Breach
- What can be placed into orbit
- How Personal Mass affects movement
- What Closure consumes
- Where an ability originates
- What Dead Star becomes

---

# 2. Voidbringer equipment layout

The Voidbringer uses the universal AbyssFall armor slots plus four class-defining slots.

## Universal equipment

- Head
- Torso
- Arms
- Legs
- Boots
- Amulet
- Two rings
- Utility charm

These carry general defense, offense, movement, survivability and broader build effects.

## Voidbringer class equipment

### Manifold Frame

The primary weapon slot.

Determines:

- Basic attack
- Weapon Power
- Attack profile
- Null Shard animation
- Closure animation
- Whether the build naturally favors melee, projectiles or anchors

Families:

- Shear Frames
- Vector Arrays
- Well Cores

### Containment Core

Controls Instability and Breach behavior.

Determines:

- Breach threshold modifications
- Instability generation patterns
- Clean Closure rewards
- Spatial Recoil consequences
- Dead Star energy handling

### Null Array

Controls the character’s hovering Null Shards.

Determines:

- Number of visible shards
- Orbit behavior
- Projectile generation
- Defensive interception
- Anchor-placement behavior
- Shard recovery after Collapse

### Harness Assembly

Controls Personal Mass, survivability and the relationship between the character and the anchor network.

Determines:

- Barrier strength
- Self-anchor behavior
- Gravitic Skin interaction
- Impact resistance
- Movement under heavy Personal Mass
- Damage received during Breach

These four slots ensure that the class’s identity is not carried by one weapon alone.

# 3. Manifold Frame families

# Shear Frames

Shear Frames compress Null material into close-range weapons.

They support:

- Worldshear
- Mass Driver
- Countermass
- Gravitic Skin
- Event Step attacks
- Point-blank Closure

Their basic attacks generate slightly more Instability but provide higher Impact.

## Rift Cleaver

A heavy segmented blade that appears to skip portions of its swing.

Basic attack:

- Wide horizontal cut
- High stagger
- Adds Mass to enemies already carrying anchors
- Slower attack speed

Natural builds:

- Pressure Edge
- Living Singularity
- Impact Event
- Counterforce

## Null Talons

Several shards form claws around both hands.

Basic attack:

- Rapid alternating strikes
- Low individual Impact
- Generates Mass through repeated contact
- Strong interaction with movement speed

Natural builds:

- Kinetic Butcher
- Redline Paradox
- Bone Orbit
- High-Breach melee

## Event Maul

A crushing weapon that appears only at the moment of impact.

Basic attack:

- Slow overhead blow
- Extreme terrain-collision power
- Temporarily increases enemy weight
- Strongest against armor and stance

Natural builds:

- Mass Driver
- Unmovable Object
- Terrain collisions
- Heavy Personal Mass

## Meridian Blade

A long, narrow spatial edge formed between two shards.

Basic attack:

- Increased reach
- Cuts across Fold Lines
- Lower stagger
- Strong repeated-line interactions

Natural builds:

- Black Meridian
- Open Wound
- Closed Geometry
- Hybrid melee-control builds

# Vector Arrays

Vector Arrays launch, redirect and orbit Null Shards.

They support:

- Null Shard
- Vector Salvo
- Trajectory Theft
- Rail Collapse
- Periapsis
- Law of Orbit

Their basic attacks are ranged but become strongest when interacting with Fold Lines.

## Needle Array

Fires rapid, narrow Null fragments.

Basic attack:

- Fast projectile stream
- Low Impact
- Excellent Fold Line scaling
- Weak without geometry

Natural builds:

- Event Lance
- No Rest Frame
- Redline Paradox
- Fold Line rail builds

## Orbital Array

Creates projectiles that circle the nearest anchor before release.

Basic attack:

- Slower initial output
- Gains power over time
- Can maintain several orbiting shots
- Vulnerable to disruption

Natural builds:

- Orbital Execution System
- Funeral Orbit
- Law of Orbit
- Hostile Constellation

## Vector Cannon

Forms one dense projectile from several shards.

Basic attack:

- Slow firing rate
- High penetration
- Strong Velocity Reserve scaling
- High Instability per shot

Natural builds:

- Rail Collapse
- Single Vector
- Fold Rail
- Projectile artillery

## Scatter Manifold

Launches shards in a close fan.

Basic attack:

- Strong at medium range
- Loads several enemy anchors
- Weak precision at distance
- Excellent against groups

Natural builds:

- Vector Salvo
- Murder Orbit
- Carrier Brand
- Mobile pack-clearing builds

# Well Cores

Well Cores keep Null Shards detached and strengthen the anchor network.

Their basic attack is generated from the nearest active anchor rather than always from the player.

They support:

- Mass Brand
- Convergence
- Crush Point
- Accretion Field
- Black Meridian
- Dead Star

## Accretion Core

Basic attack:

- Sends a pulse from the nearest anchor
- Pulls enemies slightly
- Adds a small amount of Mass
- Low direct damage

Natural builds:

- Black Sun Architect
- Common Grave Engine
- Accretion Field
- Anchor-loading builds

## Collapse Core

Basic attack:

- Marks an anchor for increased Closure damage
- Repeated attacks build detonation pressure
- Limited personal defense

Natural builds:

- Crush Point
- Ordered Collapse
- One Final Point
- Breach resolution

## Geometry Core

Basic attack:

- Fires along Fold Lines
- Can originate from multiple anchors
- Weak without a complete network

Natural builds:

- Closed Geometry
- Murder Polygon
- Black Meridian
- Three-anchor architecture

## Vacancy Core

Basic attack:

- Creates small temporary pull regions
- Does not add much Mass directly
- Excels at repositioning enemies

Natural builds:

- Vacancy Well
- Hard Vacuum
- Nothing Escapes
- Control-focused builds

# 4. Frame statistics

Every Manifold Frame has five primary statistics.

## Weapon Power

Base damage scaling.

## Impact

Ability to stagger, launch, interrupt and cause collisions.

## Vector Control

Strength of:

- Projectile curvature
- Enemy redirection
- Orbit stability
- Fold Line transfer accuracy

## Manifold Stability

Reduces:

- Instability spikes
- Spatial Recoil severity
- Anchor disruption
- Uncontrolled orbital release

## Spatial Reach

Improves:

- Anchor placement distance
- Event Step range
- Fold Line maximum length
- Projectile interaction range

Different frames have different natural profiles.

A Rift Cleaver may have high Weapon Power and Impact but low Vector Control.

A Geometry Core may have high Vector Control and Reach but low direct Weapon Power.

# 5. Class-specific equipment identity

# Containment Cores

Containment Cores determine how aggressively the player can stress reality.

## Stable Core

- Lower Instability generation
- Stronger Clean Closure barriers
- Reduced Spatial Recoil
- Less Breach damage

Best for:

- Learning the class
- Defensive builds
- Slow setup builds
- Hardcore modes

## Overclocked Core

- Higher Instability generation
- Longer Breach potential
- Stronger high-Instability bonuses
- More severe Spatial Recoil

Best for:

- Breach Engine
- Redline
- Aggressive hybrid builds

## Recirculating Core

- Closure transfers some released Instability into the next anchor as Mass
- Breach is easier to extend
- Harder to fully return to low Instability

Best for:

- Chain Closure
- Event Horizon networks
- Long Breaches

## Inverted Core

- Spatial Recoil grants a temporary offensive benefit
- Clean Closure grants less defense
- Failed Breaches become recoverable attack windows

Best for:

- Law of Inversion
- High-risk builds
- Players who deliberately weaponize failure

# 6. Affix philosophy

An item should normally contain a mixture of:

- Foundation affixes
- Mechanic affixes
- Skill affixes
- One build-defining effect at higher rarity

## Foundation affixes

These support basic character viability:

- Maximum health
- Barrier strength
- Armor
- Elemental resistance
- Movement speed
- Weapon Power
- Critical chance
- Critical damage
- Cooldown recovery
- Healing received

## Voidbringer mechanic affixes

These support the class directly:

- Anchor duration
- Mass generated from collision
- Closure Instability removal
- Fold Line damage
- Fold Line maximum length
- Instability generated by movement skills
- Breach duration
- Clean Closure barrier
- Personal Mass maximum
- Velocity Reserve duration
- Orbit capacity
- Projectile capture capacity
- Forced movement strength
- Terrain-collision damage

## Conditional affixes

These reward specific play:

- Damage against enemies moving toward an anchor
- Damage against enemies moving away from the player
- Barrier after collapsing a Critical anchor
- Movement speed after crossing a Fold Line
- Impact after preventing displacement
- Damage while above 80 Instability
- Cooldown recovery after Clean Closure
- Increased Mass from elite deaths
- Projectile damage after one full orbit
- Melee damage per 25 Personal Mass
- Closure damage while no terrain anchor exists

## Skill affixes

Examples:

- Additional Mass Brand charge
- Event Step range
- Worldshear seam duration
- Hard Vacuum storage
- Convergence collision power
- Crush Point armor penetration
- Tidal Lock orbit speed
- Velocity Theft Reserve
- Periapsis exit velocity
- Rail Collapse penetration
- Countermass timing window
- Zero-Range Collapse pull strength

Skill affixes should improve a function, not merely add skill ranks.

---

# 7. Affix exclusions

Some affixes should not appear together naturally because the combination would eliminate meaningful tradeoffs.

## Stability versus Overclocking

An item cannot normally roll both:

- Reduced Instability generation
- Increased damage above high Instability

## Orbit versus Direct Velocity

An item cannot normally roll both:

- Increased orbit duration
- Increased direct-projectile damage based on unused orbit time

## Personal Mass versus unrestricted speed

An item cannot normally roll both:

- Greatly increased maximum Personal Mass
- No movement penalty from Personal Mass

## Anchor duration versus rapid replacement

An item cannot normally roll both:

- Greatly increased anchor duration
- Major bonuses for replacing anchors rapidly

Unique items may break these rules deliberately, but normal loot should preserve choices.

---

# 8. Legendary effect system

Legendary effects are called **Manifold Distortions**.

A Distortion changes a relationship within the Voidbringer’s system.

Most may be extracted and installed into a compatible item through the endgame Manifold Chamber.

A character may equip several Distortions, but only:

- One Primary Distortion
- Two Major Distortions
- Any number of Minor Distortions allowed by item rarity

This stops every build from stacking six reality-breaking effects simultaneously.

---

# 9. Primary Distortions

Primary Distortions define entire builds.

## The Last Orbit

Compatible slots:

- Manifold Frame
- Null Array

Effect:

- Anchors can no longer attach to terrain.
- Terrain-targeted Mass Brand creates an orbiting anchor around the Voidbringer.
- Orbiting anchors maintain different radii.
- Fold Lines rotate with the player.
- Event Step can jump to the opposite side of an orbit.

Build created:

- Mobile orbital weapons platform
- Strong with Vector Salvo and Trajectory Theft
- Weak for fixed battlefield control

## Grave of Distance

Compatible slots:

- Manifold Frame
- Harness

Effect:

- Event Step no longer moves the Voidbringer to an anchor.
- Instead, the space between the player and anchor is compressed toward the player.
- Enemies, projectiles and loose objects along the path are dragged inward.
- Event Step becomes a control skill rather than mobility.

Build created:

- Stationary spatial executioner
- Strong with Living Singularity and Hard Vacuum
- Sacrifices escape mobility

## Inward-Facing World

Compatible slots:

- Containment Core
- Well Core

Effect:

- All base pull effects become repulsions.
- Repulsed enemies gain temporary Mass based on distance traveled.
- Striking terrain causes them to implode.
- Law of Inversion instead restores normal pulls during Breach.

Build created:

- Terrain-collision artillery
- Reverses the expected class loop
- Excellent with Mass Driver and Rail Collapse

## The Empty Throne

Compatible slots:

- Well Core
- Containment Core

Effect:

- Maximum anchors become one.
- The anchor receives five Mass stages.
- Fold Line skills instead connect the player to the anchor.
- Closure at Impossible Mass becomes a major encounter-level event.

Build created:

- One Final Point without requiring the capstone
- Can combine with another discipline’s capstone
- Massive single-target setup

## Event Cannibal

Compatible slots:

- Containment Core

Effect:

- Closure no longer removes Instability.
- The nearest surviving anchor absorbs:
  - 70% of the collapsed anchor’s Mass
  - Its current temporary effects
  - One compatible orbiting object
- Breach becomes significantly harder to resolve.

Build created:

- Delayed super-collapse
- Long Breach chaining
- Dangerous failure potential

## Unfallen Weight

Compatible slots:

- Harness
- Shear Frame

Effect:

- The Voidbringer cannot be forcibly displaced.
- Every prevented displacement becomes Personal Mass.
- Event Step generates 50% more Instability.
- At maximum Personal Mass, melee attacks release stored force.

Build created:

- Counterforce tank
- Boss-impact absorber
- Strong but increasingly unstable

## Hostile Constellation

Compatible slots:

- Vector Array
- Null Array

Effect:

- Captured projectiles cannot orbit the player.
- They are distributed among active anchors.
- Every completed orbit loads Mass.
- Closure releases the entire constellation before the anchor implodes.

Build created:

- Projectile-capture network
- Excellent against ranged enemies
- Requires maintaining multiple anchors

## Corpse Geometry

Compatible slots:

- Well Core
- Null Array

Effect:

- Every enemy killed while anchored leaves a temporary corpse anchor.
- Corpse anchors form Fold Lines with living enemy anchors.
- Corpse anchors cannot be manually moved.
- They collapse automatically when their duration ends.

Build created:

- Death-chain battlefield architecture
- Strong pack clearing
- Requires careful timing against bosses

## No Safe Frame

Compatible slots:

- Redline Harness
- Containment Core

Effect:

- Instability can no longer fall below 50 during combat.
- At 100 Instability, Breach does not trigger automatically.
- The player manually activates Breach through Closure.
- Waiting too long beyond 100 begins damaging the containment barrier.

Build created:

- Manual Breach timing
- Extremely high skill ceiling
- Allows planned boss burst windows

## World Without Rest

Compatible slots:

- Vector Array
- Redline Harness

Effect:

- Standing still causes damage and projectile speed to decrease.
- Continuous movement builds Relative Velocity.
- At maximum Relative Velocity:
  - Null Shards originate from recent movement positions
  - Event Step leaves additional echoes
  - Collisions become stronger
- Taking a heavy hit removes all Relative Velocity.

Build created:

- Constant-motion projectile fighter
- High execution demand
- Fragile when interrupted

# 10. Major Distortions

## Funeral Geometry

Corpses may receive Mass Brand.

Corpse anchors:

- Last longer
- Gain Mass from nearby deaths
- Deal increased Closure damage

## Closed Circuit

Event Step between two Critical anchors:

- Does not begin its full cooldown
- Consumes 15 Mass from each endpoint
- Can be repeated until an anchor drops below Critical

## Black Meridian Engine

Enemies dragged through Black Meridian:

- Transfer part of their movement speed into both endpoint anchors
- Increase the line’s damage temporarily

## Satellite Execution

Tidal Lock may orbit an enemy around the Voidbringer.

The orbiting body:

- Protects one side of the player
- Deals contact damage
- Can be thrown through Event Step

## Negative Impact

Countermass stores damage prevented in addition to physical Impact.

The next Mass Driver releases a portion as spatial damage.

## Vacancy Body

Hard Vacuum no longer surrounds the player.

It forms at the player’s previous location whenever Event Step is used.

## Three-Body Error

Maintaining three anchors causes one additional unstable Fold Line to connect whichever two anchors currently contain the greatest Mass.

The unstable line deals greater damage but changes endpoints dynamically.

## Terminal Brand

Mass Brand maximum charges become one.

Its anchor:

- Begins Dense
- Has increased Mass capacity
- Cannot be replaced while active
- Returns its charge only through Closure

## Blood Has Weight

Bleeding or heavily wounded enemies generate Mass while moving.

The Voidbringer does not cause the bleeding automatically; this supports group synergy and physical-damage builds.

## Folded Barrage

Vector Salvo fired through a Fold Line reappears from its opposite endpoint.

Damage is reduced slightly for each duplication.

## Crushing Arrival

Event Step ending within melee range:

- Increases Personal Mass
- Produces a small ground collapse
- Empowers the next Worldshear

## Black Starved

Dead Star gains Mass more slowly from ordinary deaths but massively more from:

- Elites
- Boss stagger
- Critical anchors
- Captured heavy projectiles

# 11. Minor Distortions

Minor Distortions support a build without defining it.

Examples:

- Collapsing a Dense anchor grants movement speed.
- Event Step refreshes one second of Gravitic Skin.
- Null Shard gains one additional Fold Line crossing.
- Mass Driver creates a Dormant terrain anchor on a wall impact.
- Hard Vacuum collects loose Null Shards.
- Clean Closure restores one Mass Brand charge.
- Periapsis causes orbiting projectiles to rotate faster.
- Countermass adds Mass to the nearest terrain anchor.
- Black Meridian remains briefly after one endpoint collapses.
- Tidal Lock grants a barrier for each completed orbit.
- Velocity Theft reduces projectile cooldowns.
- Crush Point leaves an enemy briefly weightless before imploding.
- Closure against a corpse restores a small amount of health.
- Spatial Recoil increases armor during its duration.
- Breach begins with a brief burst of movement speed.
- Worldshear can sever weak enemy projectiles.
- Rail Collapse creates a temporary Fold Line along its path.
- Gravitic Clamp-held enemies add to Personal Mass.
- Dead Star detonation refreshes Event Step.

# 12. Unique artifacts

Unique items cannot be extracted into Distortions.

They are complete objects with strong identity, drawbacks and visual changes.

## 1. Axiom, the First Nail

Type:

- Unique Well Core

Appearance:

A single black spike surrounded by distorted copies of itself.

Effect:

- Maximum anchors become one.
- The anchor has no duration.
- Every ability can load it beyond normal maximum Mass.
- Mass above 100 becomes **Forbidden Mass**.
- Closure damage increases dramatically with Forbidden Mass.
- Closure permanently reduces maximum health for the remainder of the encounter based on Forbidden Mass consumed.

Purpose:

The ultimate boss-execution and risk-accumulation weapon.

The player builds one impossible point and decides how much of themselves to sacrifice when collapsing it.

## 2. Choir of Falling Iron

Type:

- Unique Null Array

Appearance:

Twelve thin shards orbit in uneven, intersecting patterns.

Effect:

- Null Shard fires from every active anchor at reduced damage.
- Shards attempt to cross at least one Fold Line before striking.
- Each shard that collides with another shard creates a micro-collapse.
- Direct Null Shard damage is reduced.

Purpose:

Turns the anchor network into a distributed firing system.

Strong with geometry, weaker as a normal ranged weapon.

## 3. Atlas of the Crushed World

Type:

- Unique Architect Harness

Appearance:

A harness engraved with shifting maps of places that no longer exist.

Effect:

- Terrain anchors inherit properties from the surface:
  - Metal adds Impact
  - Stone adds Mass capacity
  - Flesh adds healing on Closure
  - Machinery adds projectile velocity
  - Corrupted terrain adds Instability and Breach damage
- Replacing a terrain anchor erases its inherited property.

Purpose:

Rewards environmental knowledge and makes arenas mechanically relevant.

## 4. Skin of the Unmoving Saint

Type:

- Unique Siege Harness

Appearance:

Fused plates shaped like a kneeling figure wrapped around the spine.

Effect:

- Normal movement speed is heavily reduced.
- Event Step has two charges.
- Personal Mass never decays while stationary.
- At maximum Personal Mass:
  - The Voidbringer becomes immune to interruption
  - Event Step arrives with a Critical self-anchor
- Dodging removes half of Personal Mass.

Purpose:

A deliberate, heavy spatial-juggernaut build.

## 5. Red Gospel

Type:

- Unique Overclocked Core

Appearance:

A cracked core leaking narrow lines of red-shifted light.

Effect:

- Breach begins at 80 Instability.
- Breach duration is reduced.
- Every Closure during Breach causes the next ability to occur from two locations.
- Failing Clean Closure causes all echoes to repeat hostile versions of the player’s recent attacks.

Purpose:

Extreme burst potential with a brutal but readable failure state.

## 6. Teeth Around the Sun

Type:

- Unique Orbital Array

Appearance:

Curved black fragments orbit like a jaw closing around empty space.

Effect:

- Orbiting projectiles cannot be released manually.
- They release automatically when their anchor becomes Critical.
- Each released projectile bites through the anchor before firing, reducing its Mass.
- If the anchor survives, the array rebuilds the orbit.

Purpose:

Creates an automatic Mass-powered weapons cycle.

The player maintains the machine rather than manually firing it.

## 7. The Distance That Died

Type:

- Unique Meridian Blade

Appearance:

The weapon’s handle and blade appear several feet apart.

Effect:

- Worldshear does not attack in front of the player.
- It attacks along the longest active Fold Line.
- Worldshear can be used from any distance.
- The player has reduced direct melee defense.
- Cutting a Critical Fold Line briefly removes the distance between its endpoints.

Purpose:

A remote melee build that fights through geometry.

## 8. Negative Crown

Type:

- Unique Containment Core

Appearance:

A ring of fragments hovering behind the head, bending light inward.

Effect:

- Pull effects are centered behind enemies instead of at anchors.
- Enemies appear to be dragged away from the anchor while actually moving toward an invisible negative point.
- Collisions near the false center create Mass at the real anchor.
- Anchor indicators are harder to read but Collapse damage is increased.

Purpose:

A high-mastery misdirection and reverse-positioning build.

## 9. Coffin of Momentum

Type:

- Unique Vector Cannon

Appearance:

A long rectangular weapon containing a projectile frozen mid-launch.

Effect:

- Basic attacks do not fire immediately.
- Each attack input stores one shot and its current movement velocity.
- Event Step or Periapsis releases every stored shot from previous player positions.
- Taking heavy damage destroys one stored shot.

Purpose:

A choreographed burst build using movement history.

## 10. The Witnessing Mass

Type:

- Unique Harness Assembly

Appearance:

Small human-shaped impressions press outward from within the armor.

Effect:

- Nearby living enemies contribute Personal Mass.
- Nearby corpses contribute twice as much.
- Personal Mass increases damage reduction but also increases Instability generated.
- At maximum Mass, faint copies of nearby enemies orbit the player and repeat their most recent movement when Closure is used.

Purpose:

A terrifying crowd-fed Hollow Form build.

## 11. Eventide Engine

Type:

- Unique Recirculating Core

Appearance:

A mechanical core whose interior appears to rotate backward.

Effect:

- Clean Closure does not end Breach normally.
- It resets Breach to four seconds and increases Instability generation.
- Each reset adds one stack of Eventide.
- Eventide strengthens all spatial effects.
- When Breach finally ends, every Eventide stack worsens Spatial Recoil.

Purpose:

Potentially indefinite Breach for players who can keep feeding and collapsing anchors.

## 12. The World’s Missing Rib

Type:

- Unique Rift Cleaver

Appearance:

A curved black structure shaped like part of an enormous skeleton.

Effect:

- Worldshear removes a thin section of space.
- Enemies struck cannot cross the removed section normally.
- Event Step can travel through removed sections.
- Up to three sections can exist.
- Creating a fourth violently reconnects the oldest section.

Purpose:

Turns melee attacks into temporary terrain editing.

## 13. God’s Counterweight

Type:

- Unique Event Maul

Appearance:

One half of the weapon is impossibly large; the other appears almost weightless.

Effect:

- Mass Driver places equal but opposite Mass on:
  - The primary target
  - The Voidbringer
- The next movement by either body drags the other.
- If both remain stationary, the space between them collapses.
- Boss resistance strengthens the collapse rather than negating it.

Purpose:

A boss-dueling weapon built around shared movement and restraint.

## 14. Funeral Satellite

Type:

- Unique Satellite Array

Appearance:

A sealed corpse-shaped object orbiting within a ring of shards.

Effect:

- One corpse can be stored permanently in orbit.
- The corpse’s original enemy type determines a passive trait.
- Tidal Lock launches or recalls the corpse.
- Closure destroys it for a massive effect, requiring a new corpse afterward.

Possible traits:

- Heavy corpse: increased Impact
- Armored corpse: projectile defense
- Explosive corpse: detonation
- Parasite corpse: infection spread
- Caster corpse: captured projectile amplification

Purpose:

Creates a grim corpse-companion system without becoming a summoner.

## 15. Zero Is a Place

Type:

- Unique Vacancy Core

Appearance:

A hollow object whose center displays no background at all.

Effect:

- Vacancy effects become targetable locations.
- Event Step can travel to a Vacancy.
- Mass Brand can anchor a Vacancy.
- Collapsing a Vacancy temporarily removes enemies inside from the battlefield.
- Removed enemies return violently after a delay.

Purpose:

A control build based on weaponized absence.

# 13. Unique item acquisition philosophy

The most transformative items should come from content related to their fantasy.

Examples:

- **Axiom, the First Nail:** Dropped by a boss built around one immovable central object.
- **Atlas of the Crushed World:** Earned from environmental-collapse challenges.
- **Coffin of Momentum:** Dropped by a boss whose attacks remain suspended in time.
- **Funeral Satellite:** Earned from a corpse-gravity monstrosity.
- **The World’s Missing Rib:** Found inside a region where sections of terrain have vanished.
- **Eventide Engine:** Reward for maintaining an extended Breach during a class trial.

This makes loot feel discovered rather than randomly dispensed.

# 14. Manifold Imprints

Unlocked after the final level-50 class trial.

An Imprint is a high-end modification installed into one class item.

Each item may hold one Imprint.

Imprints do not add entirely new systems. They deepen existing ones.

## Imprint of the Third Point

The third anchor placed always begins with Mass equal to the average of the first two anchors.

## Imprint of Closure Memory

Closure remembers the previous anchor type.

Collapsing a different carrier type next grants a bonus:

- Enemy after terrain: stronger direct damage
- Terrain after enemy: larger area
- Corpse after self: healing
- Self after corpse: Personal Mass

## Imprint of Repeated Distance

The first Event Step after Clean Closure leaves a destination echo usable for a second free return.

## Imprint of Witnessed Momentum

Enemy movement abilities seen during Breach increase Velocity Theft effectiveness against that enemy type.

## Imprint of Dense Silence

Critical anchors suppress enemy audio and visual telegraphs less clearly but significantly reduce enemy ability speed.

This is powerful but dangerous because it also makes attacks harder to read.

## Imprint of Forbidden Geometry

Four anchors may exist, but no automatic Fold Lines form.

The player creates one line at a time by targeting its endpoints.

This is intended only for high-level players who want manual geometry control.

# 15. Crafting rules

Voidbringer crafting should support targeted experimentation without guaranteeing perfect items instantly.

## Recalibration

Change one numerical affix while preserving the rest of the item.

## Vector Realignment

Change one mechanic affix into another from the same category.

Example:

- Fold Line length → Fold Line damage
- Closure Instability removal → Closure radius
- Personal Mass maximum → Personal Mass decay

## Distortion Extraction

Destroy a Legendary item to extract its Distortion.

Primary Distortions require a rare class material to reinstall.

## Core Overclocking

Increase the numerical power of a Containment Core while also increasing one downside.

The player chooses the downside from a limited list.

## Frame Tempering

Improve:

- Weapon Power
- Impact
- Vector Control
- Stability
- Spatial Reach

Only two Frame statistics may be fully maximized on one item.

## Null Reassembly

Reroll the behavioral profile of a Null Array without changing its rarity.

# 16. Loot readability

An item comparison should show three layers.

## Layer one: survival and output

- Weapon Power
- Armor
- Barrier
- Resistances
- Health

## Layer two: class-system changes

- Instability
- Mass
- Closure
- Breach
- Fold Lines
- Personal Mass
- Orbit behavior

## Layer three: build consequences

The interface should summarize major behavioral changes in direct language.

Examples:

> **Your anchors can no longer attach to terrain.**

> **Event Step pulls space toward you instead of moving you.**

> **Closure no longer reduces Instability.**

> **Your projectiles must orbit before striking Critical targets.**

The player should not need a community spreadsheet to understand what an item fundamentally does.

# 19. Anti-meta design protections

Build diversity disappears when one mathematical strategy scales every mechanic at once.

The following protections should exist.

## Distinct damage scaling

Collision, projectile, Fold Line, Collapse and melee damage should use partially separate scaling pools.

One stat should not perfectly scale all five.

## Distinct resource goals

Some builds want:

- Low Instability
- High Instability
- Frequent short Breaches
- Long Breaches
- Failed Breaches
- Rare deliberate Breaches

No single Containment Core should dominate all strategies.

## Real tradeoffs

Powerful effects remove something meaningful:

- Terrain anchors
- Mobility
- Stability
- Direct damage
- Defensive orbit
- Clean Closure
- Anchor count
- Manual control

## Capstone exclusivity

Only one Grand Capstone may be active.

A Living Singularity cannot also gain the complete benefits of Orbital Execution System and Black Sun Architect.

Hybrid keystones still provide cross-discipline creativity.

## Unique items are not mandatory upgrades

Each Unique should create a build, not become the automatic best item for every build using that skill.

A rare or Legendary item with ideal affixes should remain competitive.

# 20. Implementation requirements

Every Distortion should be represented as a modular rules change rather than custom code scattered across multiple abilities.

Examples:

## Anchor carrier rule

Allowed carriers:

- Enemy
- Terrain
- Corpse
- Player
- Orbit
- Vacancy
- Projectile

## Force-direction rule

Possible force modes:

- Pull
- Repel
- Orbit
- Hold
- Reverse
- Compress toward line
- Compress toward midpoint

## Skill-origin rule

Possible origins:

- Player
- Anchor
- Fold Line
- Echo
- Prior position
- Dead Star
- Held enemy

## Closure-output rule

Possible outputs:

- Implosion
- Projectile
- Orbit release
- Force wave
- Mass transfer
- Self-collapse
- Delayed collapse

Designing the system around these reusable rules allows new items to alter the class without requiring every skill to be rebuilt individually.

# 21. Itemization completion standard

Voidbringer loot is successful when:

- A new weapon can change the player’s basic combat rhythm.
- A Containment Core meaningfully changes Breach management.
- Null Arrays create visible differences in shard behavior.
- Harnesses differentiate mobile, defensive and self-anchor builds.
- Legendary effects alter mechanics rather than only damage.
- Unique items create strange but coherent new playstyles.
- Rare items remain valuable through strong affix combinations.
- No single statistic scales every build.
- Gear supports all three disciplines and their hybrids.
- Players can recognize another Voidbringer’s build by watching it fight.

The final loot fantasy is:

> **The Voidbringer is not collecting stronger weapons. They are replacing the laws by which their containment device is allowed to operate.**
