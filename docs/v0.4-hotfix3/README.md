# AbyssFall: Void Warlock v0.4 — The Sunken Crypts

This build turns the arena prototype into the first complete level slice: a room-by-room descent through the Sunken Crypts, ending in the first major boss fight.

## Open and run

1. Extract the ZIP.
2. Open Godot 4.2 or newer.
3. Import `project.godot`.
4. Press **F5**.

## Controls

### Keyboard and mouse

- **WASD** — Move
- **Mouse** — Aim
- **Left click** — Void Bolt
- **Right click / Q** — Grasping Rift
- **Space / Shift** — Shadow Step
- **E** — Interact with the hidden reliquary
- **I** — Inventory
- **T** — Skill tree
- **1 / 2 / 3** — Select a level-up
- **Escape** — Close menus
- **R** — Restart

### Controller

- **Left stick** — Move
- **Right stick** — Aim
- **Right bumper** — Void Bolt
- **Left bumper** — Grasping Rift
- **A / Cross** — Shadow Step
- **B / Circle** — Interact
- **Y / Triangle** — Inventory
- **X / Square** — Skill tree

## Level route

1. Ritual Courtyard — destroy two Abyss Generators.
2. Generator Chamber — destroy the deep generator while fighting ranged defenders and an elite brute.
3. Collapsed Catacombs — survive two ambush waves.
4. Trap Hallway — avoid cycling void spikes and clear the defenders.
5. Hidden Relic Alcove — optional Epic reward.
6. Abyssal Throne — defeat the Hollow King.

## New in v0.4

- First connected multi-room level
- Gate-based room progression and objective updates
- Bone Archer ranged enemy
- Crypt Brute heavy enemy with a ground slam
- Cycling void-floor traps
- Optional hidden relic interaction
- The Hollow King boss with three phases
- Boss health UI and phase callouts
- Expanded fourteen-item loot pool
- Guaranteed Epic exploration reward
- Guaranteed Legendary boss reward: Crown of the Hollow King

## Prototype note

All environments and characters are still generated from placeholder geometry. The goal is to test pacing, room flow, enemy roles, loot progression, and the boss encounter before importing final models and animation.
