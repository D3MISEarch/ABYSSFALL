# AbyssFall Development Roadmap

## v0.4 — The Sunken Crypts (Current)

- Connected first level
- Three enemy roles
- Environmental traps
- Optional relic room
- Three-phase Hollow King boss
- Expanded loot and boss rewards

## v0.4.1 — Runtime Playtest and Tuning

- Fix Godot runtime errors
- Tune room pacing and enemy counts
- Balance XP, drops, Rift cost, and boss damage
- Improve gate and objective clarity
- Add temporary sound effects

## v0.5 — Presentation and Content

- Replace placeholders with first imported character/enemy models
- Animation state machines
- Stronger VFX and impact feedback
- Sound identity and music layers
- First hub prototype and portal selection
- Save data for persistent unlocks

## v0.6 — Meta Progression

- Run completion rewards
- Relic codex
- Permanent unlock tree
- Difficulty modifiers
- Additional Warlock abilities

## v0.7 — Second Realm and New Class

- New level theme and enemy faction
- Second boss
- Second playable class
- Expanded item pool and class-specific gear

## v0.8 — Local Co-op Foundation

- Two-player camera and spawn flow
- Shared-room encounter rules
- Revive system
- Co-op loot behavior

## v0.9 — Online Networking Prototype

- Authority model
- Lobby and session flow
- State replication
- Disconnect/rejoin handling

## v1.0 — Initial Release Target

- Multiple realms and bosses
- Several playable classes
- Finalized progression and inventory
- PC and Android optimization
- Local and online multiplayer
